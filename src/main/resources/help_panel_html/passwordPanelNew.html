Apply For And Manage Your Certificate(s)

This part of the tool is for managing your certificates. You can:
  • Apply for UK e-Science certificate(s)
  • Import your existing certificates for subsequent renewal, installation and revocation
  • Renew your certificates
  • Install a certificate as .pem files for subsequent Grid authentication
  • Submit revocation requests to disable certificates

- Your certificates are stored in a password-protected Keystore file ('$HOME/.ca/cakeystore.pkcs12').
- This file can be copied/backed-up to different computers if required.
